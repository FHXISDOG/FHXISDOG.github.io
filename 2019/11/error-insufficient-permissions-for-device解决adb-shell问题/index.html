Error: Insufficient Permissions for Device(解决adb Shell问题)

2019-11-10 flutter flutter android 评论 字数统计:335字 阅读时长:1分

在linux连接android手机，使用adb出现error: insufficient permissions for device

输入 adb-devices显示:

List of devices attached
6180bb53	no permissions; see [http://developer.android.com/tools/device.html]

解决方法:

1.首先在终端查看usb的ID，输入lsusb命令，我们可以看到我们刚插如usb的ID号，如：

Bus 002 Device 001: ID 1d6b:0003 Linux Foundation 3.0 root hub
Bus 001 Device 005: ID 5986:2113 Acer, Inc 
Bus 001 Device 006: ID 8087:0a2a Intel Corp. 
Bus 001 Device 007: ID 24ae:1813  
Bus 001 Device 024: ID 05c6:676c Qualcomm, Inc.    //这个就是我的设备
Bus 001 Device 017: ID 0c45:8073 Microdia 
Bus 001 Device 001: ID 1d6b:0002 Linux Foundation 2.0 root hub

2.那么我们进入到cd /etc/udev/rules.d/下，新建一个51-android.rules文件（sudo vim 51-android.rules），在这个文件中写上：

SUBSYSTEM=="usb", ATTRS{idVendor}=="05c6", ATTRS{idProduct}=="676c",MODE="0666"

保存，再为51-android.rules加上权限（sudo chmod a+x 51-android.rules）.

3.拔掉usb重新插上就可以了
