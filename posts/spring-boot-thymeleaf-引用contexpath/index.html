Spring Boot Thymeleaf 引用contexpath

04 Sep 2019, 16:03
spring boot
java / spring boot

spring boot 设置contextPath后，如果js中的请求路径不设置，会造成404的问题。 设置方法:

在html页面header中添加配置:

<meta name="ctx" th:content="${#httpServletRequest.getContextPath()}" >

js中引用:

 var ctx = $("meta[name='ctx']").attr("content");

变量ctx就是contextPath
